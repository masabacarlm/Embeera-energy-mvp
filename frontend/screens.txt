Embeera Energy Mobile App Demo Screens
======================================

This classroom demo is implemented in frontend/App.js as one simple scrollable
React Native Expo screen. Complex navigation is intentionally not included yet.

1. Welcome Screen

Purpose:
Introduces Embeera Energy as a clean cooking savings and health-tech MVP.

Content:
- Embeera Energy MVP badge
- Tagline: Saving Together. Living Better.
- Short clean cooking explanation
- Register and Login demo buttons

2. Registration Screen Placeholder

Purpose:
Shows what a future household registration form will collect.

Content:
- Full name
- Phone number
- District or location
- Household size
- Preferred access method: App or USSD

3. Dashboard Showing Savings Progress

Purpose:
Shows the user's savings journey toward an LPG starter kit.

Content:
- Amount saved
- Savings target
- Progress percentage
- Simple progress bar
- Circle, rewards points, and progress summary

4. Oluganda Circle Screen

Purpose:
Shows group-based savings for households transitioning to LPG.

Content:
- Circle name
- Number of members
- Total circle savings
- Shared LPG starter kit goal

5. Mock Payment Screen

Purpose:
Demonstrates a simple mobile money savings payment.

Content:
- Demo amount
- MTN MoMo option
- Airtel Money option
- Successful demo transaction status

6. Learning and LPG Safety Screen

Purpose:
Teaches clean cooking benefits and safe LPG use.

Content:
- Ventilation
- Cylinder storage
- Leak checks
- Safe stove use
- Lesson completion progress

7. Rewards and Enkola Certificate Screen

Purpose:
Shows incentives for saving, learning, referrals, and safe LPG adoption.

Content:
- Reward points
- Referral rewards
- Savings rewards
- Enkola Certificate progress

8. LPG Delivery Tracking Screen

Purpose:
Tracks delivery of the LPG starter kit.

Content:
- Stove and cylinder delivery
- Regulator and safety card
- Delivery status
- Current delivery stage

Visual Style
------------

- Medical blue
- Teal
- White background
- Rounded cards
- Readable text
- Simple beginner-friendly React Native code
