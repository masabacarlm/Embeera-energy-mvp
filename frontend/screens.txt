Embeera Energy Web Frontend
===========================

This frontend is a React web dashboard served with Vite. The interface uses
Bootstrap for the responsive container, rows, columns, and spacing, with
Material UI components for form controls, cards, buttons, alerts, chips, and
progress indicators.

Main dashboard sections
-----------------------

1. Top Navigation

- Embeera Energy brand
- Mukono status chip
- Profile refresh action

2. Household Profile

- Amina Nakato
- Mukono
- Household size and active Oluganda Circle
- LPG Stove and Cylinder goal

3. Savings Progress

- UGX amount saved
- UGX target and remaining balance
- Progress bar toward the LPG savings goal

4. Oluganda Circle

- Mukono Clean Cooking Group
- Seeta LPG Savings Circle
- Group joining action using POST /api/groups/join

5. Save Money

- UGX amount field
- MTN MoMo and Airtel Money payment options
- Payment action using POST /api/payments/mock

6. Learning Checklist

- LPG benefits
- LPG safety tips
- Clean cooking transition checklist
- Learning update action using POST /api/learning/update

7. Rewards and Enkola Certificate

- Reward points
- Certificate status
- Certificate readiness checklist

8. LPG Delivery Tracking

- LPG Stove and Cylinder delivery request
- Mukono delivery location
- Delivery request action using POST /api/deliveries/request

9. Admin Overview

- Household count
- Active Oluganda Circles
- Savings recorded
- Pending deliveries
- Certificates and ambassadors
