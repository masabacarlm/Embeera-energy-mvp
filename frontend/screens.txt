Embeera Energy Web Frontend
===========================

This frontend is a React web dashboard served with Vite. The interface uses
Bootstrap for the responsive container, rows, columns, and spacing, with
Material UI components for form controls, cards, buttons, alerts, chips, and
progress indicators.

Main dashboard sections
-----------------------

1. Authentication

- Embeera Energy brand
- Sign in with phone number
- Create account with name, phone, location, and role
- Oluganda Circle clean cooking message

2. Top Navigation

- Embeera Energy brand
- Signed-in user chip
- Role chip
- Logout action

3. Member Dashboard

- UGX amount saved
- UGX target and remaining balance
- Progress bar toward the LPG savings goal

4. Oluganda Circle

- Mukono Clean Cooking Group
- Seeta LPG Savings Circle
- Circle creation and invite-code joining

5. Save Money

- UGX amount field
- MTN MoMo and Airtel Money payment options
- Contribution action linked to the active circle

6. Learning Checklist

- Clean LPG readiness lessons
- Lesson completion action

7. Rewards and Enkola Certificate

- Certificate status
- Certificate readiness checklist

8. Ambassador Dashboard

- Referral form
- Household transition progress

9. Admin Overview

- Household count
- Active Oluganda Circles
- Savings recorded
- Pending deliveries
- Certificates and ambassadors
