Embeera Energy User Journey
===========================

Purpose
-------

This document explains how a household moves through the Embeera Energy MVP from registration to clean-energy transition.

Main User Journey
-----------------

1. User discovers Embeera Energy
   - Through an Omukwano Ambassador, church, NGO partner, community group, or mobile campaign.

2. User registers
   - The user provides name, phone number, location, and household details.

3. User joins an Oluganda Circle
   - The user joins a community savings group focused on LPG transition.

4. User starts saving
   - The user saves money using mock MTN MoMo, mock Airtel Money, or the simulated USSD flow.

5. System tracks savings progress
   - The user sees amount saved, savings target, progress percentage, and remaining balance.

6. User learns about clean cooking
   - The user accesses LPG safety tips and clean cooking education.

7. User qualifies for LPG transition
   - Once the savings target is reached and learning is completed, the user becomes ready for delivery.

8. User requests LPG delivery
   - The system creates a delivery request.

9. Delivery is tracked
   - Delivery status moves from Pending to In Transit to Delivered.

10. User receives Enkola Certificate
    - After delivery is completed, the household receives the certificate.

11. User earns rewards
    - The user earns reward points for saving, learning, referrals, and completing the transition.

Simple Flow
-----------

Register
? Join Oluganda Circle
? Save Money
? Track Progress
? Learn
? Request LPG Delivery
? Receive LPG Equipment
? Receive Enkola Certificate
? Earn Rewards
