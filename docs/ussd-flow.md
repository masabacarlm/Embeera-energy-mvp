Embeera Energy USSD Flow
========================

Purpose
-------

The USSD flow allows users without smartphones or internet access to access Embeera Energy using the short code *284*88#.

USSD Short Code
---------------

*284*88#

Main Menu
---------

Welcome to Embeera Energy
Saving Together. Living Better.

1. Join Oluganda Circle
2. Save Money
3. Check Savings Progress
4. Learn About Clean Energy
5. View Rewards
6. Request LPG Delivery
7. Contact Ambassador

Option 1: Join Oluganda Circle
------------------------------

User selects:
1. Join Oluganda Circle

System asks:
Enter your location

User enters:
Mukono

System responds:
Available groups in Mukono:
1. Mukono Clean Cooking Group
2. Seeta LPG Savings Circle

User selects a group.

System responds:
You have successfully joined the selected Oluganda Circle.

Option 2: Save Money
--------------------

User selects:
2. Save Money

System asks:
Enter amount to save

User enters:
10000

System asks:
Choose payment method:
1. MTN MoMo
2. Airtel Money

User selects payment method.

System responds:
Payment received successfully. Your savings balance has been updated.

Option 3: Check Savings Progress
--------------------------------

User selects:
3. Check Savings Progress

System responds:
Amount Saved: UGX 80,000
Savings Target: UGX 250,000
Progress: 32%
Remaining: UGX 170,000

Option 4: Learn About Clean Energy
----------------------------------

User selects:
4. Learn About Clean Energy

System shows:
1. Benefits of LPG
2. LPG Safety Tips
3. Why Reduce Charcoal and Firewood Use
4. Transition Checklist

Option 5: View Rewards
----------------------

User selects:
5. View Rewards

System responds:
Reward Points: 120
Enkola Certificate Status: In Progress

Option 6: Request LPG Delivery
------------------------------

If savings target is not reached:

System responds:
You have not yet reached your LPG savings target.

If savings target is reached:

System responds:
Your LPG delivery request has been created.
Status: Pending

Option 7: Contact Ambassador
----------------------------

User selects:
7. Contact Ambassador

System responds:
Your Omukwano Ambassador will contact you for support.
