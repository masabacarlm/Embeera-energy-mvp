Embeera Energy MVP User Story
=============================

Purpose
-------

This document explains the main MVP journey for Embeera Energy. It shows how one household moves from registration to saving, learning, delivery, certification, and rewards.

Story Title
-----------

From Charcoal to LPG Through Community Saving

Seeded User
-----------

Name:
Amina Nakato

Location:
Mukono

User Type:
Household

Savings Group:
Mukono Clean Cooking Group

Savings Target:
UGX 250,000

Starting Savings:
UGX 80,000

MVP Journey
-----------

Step 1: User Discovers Embeera Energy

Amina hears about Embeera Energy from an Omukwano Ambassador in Mukono. The ambassador explains that Embeera helps households save gradually toward LPG cooking equipment.

Step 2: User Registers

Amina registers on Embeera Energy using her name, phone number, and location.

Step 3: User Joins an Oluganda Circle

Amina joins the Mukono Clean Cooking Group, which is an Oluganda Circle created to help households save together toward LPG transition.

Step 4: User Saves Money

Amina saves UGX 10,000 using the local simulated MTN MoMo payment option.

The system saves the transaction in MySQL and updates savings progress.

Step 5: System Updates Savings Progress

The system updates Amina's savings progress toward the LPG target.

Step 6: User Learns About Clean Cooking

Amina completes learning topics about LPG benefits, LPG safety, and clean cooking transition readiness.

Step 7: User Requests LPG Delivery

After joining a circle, Amina can request LPG stove and cylinder delivery.

Step 8: Delivery Is Tracked

The delivery request is stored with a Pending status for admin follow-up.

Step 9: User Receives Enkola Certificate

After savings, learning, and delivery conditions are met, the user can become eligible for an Enkola Certificate.

Step 10: Admin Reviews Progress

The admin overview shows households, active groups, total savings, pending deliveries, issued certificates, and active ambassadors from MySQL.

MVP Flow
--------

Register
Join Oluganda Circle
Save Money
Track Progress
Learn
Request LPG Delivery
View Enkola Certificate Status
Review Admin Overview

Conclusion
----------

The MVP shows that Embeera Energy can support households that cannot afford LPG upfront by helping them save together, learn, transition, and receive clean cooking equipment through a structured community-based system.
