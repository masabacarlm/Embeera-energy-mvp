Embeera Energy Demo Story
=========================

Purpose
-------

This document explains the demo journey for the Embeera Energy MVP. It shows how one household moves from registration to saving, learning, delivery, certification, and rewards.

Demo Title
----------

From Charcoal to LPG Through Community Saving

Demo User
---------

Name:
Amina Nakato

Location:
Mukono

User Type:
Household

Savings Group:
Mukono Clean Cooking Group

Savings Target:
UGX 250,000

Starting Savings:
UGX 80,000

Demo Journey
------------

Step 1: User Discovers Embeera Energy

Amina hears about Embeera Energy from an Omukwano Ambassador in Mukono. The ambassador explains that Embeera helps households save gradually toward LPG cooking equipment.

Step 2: User Registers

Amina registers on Embeera Energy using her name, phone number, and location.

Sample user details:

Full Name:
Amina Nakato

Phone Number:
0700000001

Location:
Mukono

User Type:
Household

Step 3: User Joins an Oluganda Circle

Amina joins the Mukono Clean Cooking Group, which is an Oluganda Circle created to help households save together toward LPG transition.

Step 4: User Saves Money

Amina saves UGX 10,000 using the mock MTN MoMo payment option.

The system responds:

Payment successful. Your savings balance has been updated.

Step 5: System Updates Savings Progress

The system updates Amina's savings progress.

Amount Saved:
UGX 80,000

Savings Target:
UGX 250,000

Progress:
32%

Remaining Amount:
UGX 170,000

Step 6: User Learns About Clean Cooking

Amina completes learning topics about:

- Benefits of LPG
- LPG safety tips
- Reducing charcoal and firewood use
- Clean cooking transition checklist

Step 7: User Requests LPG Delivery

After reaching the required savings target, Amina requests LPG stove and cylinder delivery.

The system creates a delivery request.

Delivery Status:
Pending

Step 8: Delivery Is Tracked

The delivery status moves through these stages:

- Pending
- In Transit
- Delivered
- Completed

Step 9: User Receives Enkola Certificate

After the LPG equipment is delivered and the transition is confirmed, Amina receives an Enkola Certificate.

Certificate Status:
Issued

Step 10: User Earns Rewards

Amina earns reward points for:

- Saving money
- Completing learning
- Transitioning to LPG
- Participating in Embeera Energy

Demo Conclusion
---------------

The demo proves that Embeera Energy can support households that cannot afford LPG upfront by helping them save together, learn, transition, and receive clean cooking equipment through a structured community-based system.
