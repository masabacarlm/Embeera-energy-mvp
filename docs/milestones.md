Embeera Energy MVP Milestones
=============================

M1: Project Setup and MVP Scope
-------------------------------

Define the MVP scope, create the repository, set up the Kanban board, assign roles, and agree on the first version of the product.

M2: User Journey and System Flow
--------------------------------

Map the full household journey from joining Embeera, joining an Oluganda Circle, saving money, learning about clean energy, transitioning to LPG, and earning rewards.

M3: Database Design and ERD
---------------------------

Design the database tables, relationships, primary keys, foreign keys, and ERD for users, groups, savings, transactions, rewards, ambassadors, and deliveries.

M4: Backend API Development
---------------------------

Create the Node.js Express backend with API routes for registration, groups, savings, mock payments, rewards, ambassadors, and delivery tracking.

M5: Mobile App Interface
------------------------

Create the React Native mobile app screens including welcome, registration, dashboard, savings group, payment, rewards, learning, and delivery status screens.

M6: USSD Flow Simulation
------------------------

Create and test a simulated USSD menu for *284*88# with options for joining a group, saving money, checking progress, learning, and earning rewards.

M7: Savings Group Feature
-------------------------

Build the Oluganda Circle feature where users can join a group, view group members, see savings targets, check group progress, and track contributions.

M8: Mock Payment Flow
---------------------

Create mock MTN MoMo and Airtel Money payment flows that allow users to make sample savings transactions and update their savings balance.

M9: Learning and Transition Tracking
------------------------------------

Create a learning section that helps households understand clean cooking, LPG safety, savings progress, and transition readiness.

M10: Rewards and Enkola Certificate
-----------------------------------

Build the rewards system and Enkola Certificate status for households that successfully complete the clean energy transition.

M11: Ambassador Referral Feature
--------------------------------

Create the Omukwano Ambassador feature for tracking referrals, community outreach, household support, and adoption progress.

M12: LPG Delivery Tracking
--------------------------

Create delivery tracking for LPG stoves and cylinders, including Pending, In Transit, Delivered, and Completed statuses.

M13: Testing and Debugging
--------------------------

Test all MVP features, fix errors, confirm API responses, test UI screens, and ensure the user flow works.

M14: Documentation and README
-----------------------------

Write the project README, setup instructions, API documentation, database guide, team roles, and explanation of how the MVP works.

M15: Final MVP Walkthrough
--------------------------

Prepare the final MVP user story, screenshots, pitch explanation, and walkthrough showing how Embeera Energy helps households save together and transition to LPG.
